请用 C 语言实现矩阵乘法，并尽力提高其运行速度。

## 要求

1.  只需要使用 `float` 类型，因为它是深度学习中最常用的数据类型。

2.  使用简单的多重循环，以直接的方式实现一个函数 `matmul_plain()`，作为基准测试。

3.  使用 SIMD、OpenMP 或其他技术（如 Intel AMX、ARM SME）实现一个函数 `matmul_improved()` 来提高速度。你可以将它与 `matmul_plain()` 进行比较。

4.  使用 16x16、128x128、1Kx1K、8Kx8K 和 64Kx64K 的矩阵测试性能。你可以生成一些随机值的矩阵来测试你的函数。

5.  将你的实现与 OpenBLAS（https://www.openblas.net/）中的矩阵乘法进行比较。你的实现结果应该与 OpenBLAS 的结果相同或非常相似。你的实现能像 OpenBLAS 一样快吗？

6.  [可选] 在 X86 和 ARM 平台上测试你的程序，并描述差异。

## 提示

1.  你可以为矩阵定义一个结构体，例如：

```c
struct Matrix
{
    size_t rows;
    size_t cols;
    float *data;
}
```

2.  请专业地实现你的函数。至少在使用参数之前要对它们进行检查。

3.  注意内存管理。不要发生内存泄漏。

4.  测量 DEBUG 模式下生成的程序的效率是没有意义的。DEBUG 模式是许多 IDE 的默认模式。如果你使用 `-O3` 选项编译 C 程序，你会感到惊讶。
